Neural Decompiler

A neural network reads EVM bytecode and reconstructs the original source logic — then proves it's correct by re-executing the bytecode. If it can't prove it, it says so. It never returns a wrong answer dressed up as right.

Built from scratch in NumPy — own autograd, GRU encoder/decoder, attention. No PyTorch, no LLM.

Use it as an API

POST /api/decompile   body: {"bytecode":"0x60..."}  or  {"asm":"PUSH1 0x00 ..."}

Returns source, a boolean verified, and a proof array of (inputs → bytecode_out == source_out) rows. verified:false means the model refused to guess — that's the feature.
